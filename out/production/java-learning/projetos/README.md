# Projetos de lógica
****
**Objetivo:** Praticar conceitos fundamentais em 
lógica de programação com a linguagem Java. Estes incluem: 
- Variáveis e tipos primitivos;
- Estruturas de controle de fluxo;
- Operadores;
- Vetores (arrays).

****

Os códigos incorporam tratamento simples para 
garantir entradas de usuário
dentro do escopo proposto, além de apresentar um nível inicial de
modularização, utilizando métodos estáticos 
dentro da própria classe.
